# Placelytics: A Data-Driven Analysis of Student Placement Patterns

## Project Overview
Placelytics is a student placement data analysis project created to study academic and skill-based factors affecting placement outcomes.

## Dataset Features
- CGPA
- Internships
- Projects
- Certifications
- Aptitude
- Chances
- Placement Status
- Improvement

## Tools Used
- R
- Jupyter Notebook
- Linear Regression

## Project Objectives
- Analyze student placement patterns
- Study relationship between CGPA and aptitude
- Visualize student performance
- Estimate placement readiness
- Generate improvement insights

## Model Used
Linear Regression

## Output
- Histogram of CGPA
- Boxplot Analysis
- Placement Status Distribution
- Correlation Analysis
- Interactive Placement Predictor
